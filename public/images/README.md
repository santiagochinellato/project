# Instrucciones para el Logo EROSCAPE

Para que el logo se muestre correctamente en la página, necesitas copiar la imagen del logo a esta ubicación:

**Ruta requerida:** `/public/images/eroscape-logo.png`

## Pasos:

1. Guarda tu imagen del logo EROSCAPE (img_7272_2.jpeg)
2. Cópiala a esta carpeta: `/public/images/`
3. Renómbrala como: `eroscape-logo.png`

El logo aparecerá automáticamente en:
- Navbar (parte superior)
- Hero section (sección principal)
- Footer (pie de página)

**Nota:** Asegúrate de que el fondo de la imagen sea transparente (formato PNG) para que se vea mejor sobre fondos oscuros.
